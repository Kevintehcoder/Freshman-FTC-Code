# Freshman FTC Robotics Code
Code written for my first year on an FTC robotics team, 
implementing autonomous movement and shooter control.

## Tech Used
* Java
* FTC SDK
* Android Studio

## Systems Implemented
* Telemetry for real-time motor and sensor debugging
* Odometry for positional tracking and autonomous movement
* Shooter mechanism control
* Driver-controlled (TeleOp) movement

## Logic used
* Working with hardware inputs and outputs in code
* Using telemetry to debug a physical robot in real time
* Implementing odometry for autonomous positioning
